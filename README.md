# Hola Laguna





## Installation


1. Clone the repository:
   

3. Install dependencies:

    ```sh
    npm install
    ```
    
4. prisma generate:

    ```sh
    npx prisma generate 
    ```

5. Start the server:

    ```sh
    npm run dev
    ```
6. prisma studio (optional) :

    ```sh
    npx prisma studio 
    ```

## Running the Application


- The server will run on `http://localhost:3000`.
- The prisma studio server will run on `http://localhost:5555`.
